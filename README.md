# 🌎 Conversor de Moedas em Tempo Real (JavaScript + API)

Este projeto é um conversor de moedas que utiliza a API pública do ExchangeRate Host para converter valores de Real (BRL) para outras moedas, como Dólar (USD), Euro (EUR) e Peso Argentino (ARS), de forma dinâmica e em tempo real.

## 🚀 Funcionalidades

- Conversão em tempo real BRL → USD, EUR, ARS
- Atualização automática via API
- Interface interativa com HTML e JavaScript
- Layout responsivo e fácil de usar

## 🧪 Tecnologias

- HTML5
- JavaScript (ES6+)
- API: https://exchangerate.host

## 💻 Como usar

1. Salve o arquivo como `index.html`
2. Abra o arquivo com um navegador (Chrome, Edge, etc.)
3. Digite o valor em BRL, selecione a moeda e clique em **Converter**
